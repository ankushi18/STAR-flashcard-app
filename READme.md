STAR Prep Deck: Interview Flashcard App

A flashcard app for practicing behavioral interview questions using the STAR (Situation, Task, Action, Result) method. This app saves all your cards to your browser's local storage.

Features

Add New Cards: A simple form to add new behavioral questions and your corresponding STAR answers.

Persistent Storage: Saves your entire deck of cards to your browser's localStorageâ€”no database needed!

3D Card Flip: Click "Flip" to see the answer on the back of the card with a smooth 3D animation.

Deck Navigation: Use "Next" and "Previous" buttons to cycle through your deck of practice questions.

Responsive Design: Looks and works great on both desktop and mobile browsers.

Tech Stack

This project was built as a portfolio piece to practice core web development skills.

HTML5: Semantic HTML for structure and accessibility.

CSS3: Custom properties, Flexbox, Grid, and 3D transforms/transitions for a modern, responsive layout.

Vanilla JavaScript (ES6+): DOM manipulation, event listeners, and localStorage to create a fully interactive single-page application (SPA).
